# Modulo14Projeto-Rodrigo-Tiago.
Sistema de Gestão de Base de Dados de Hospital

Este repositório contém uma base de dados completa para Gestão de Hospitais, criada com MySQL e desenvolvida e testada no DBeaver.
O objetivo é demonstrar como modelar e implementar um sistema que representa a realidade de um hospital: hospitais, enfermarias, empregados, médicos, enfermeiros, doentes e análises laboratoriais. Inclui também segurança com ROLES e permissões de acesso.

O que temos ralizado neste projeto:
Script SQL de criação da Base de Dados (tabelas + chaves primárias + chaves estrangeiras)
Script de povoamento com INSERTs
Script de segurança (criação de utilizadores e permissões)
Consultas SQL obrigatórias
Consultas SQL adicionais para análise

Como utilizar este projeto no seu computador
instalar MySQL
(tem de ter MySQL Server instalado)
instalar DBeaver
(abrir DBeaver e criar uma ligação ao MySQL local)
abrir os ficheiros .sql deste repositório no DBeaver

executar pela ordem:
01_criacao_tabelas.sql
02_inserts.sql
03_roles_permissoes.sql
queries.sql
confirmar que as tabelas aparecem no esquema “Hospital”

Depois disto, a base de dados fica criada e pode executar as consultas diretamente no DBeaver.
